## 개발환경 설정: 작업 폴더 만들기

코드는 메모장의 똑똑한 버전인 VSCode(Visual Studio Code)에서 작성합니다. 글자에 색이 입혀져 코드를 훨씬 쉽게 알아볼 수 있습니다.

### 내 작업 폴더 열기 순서

- **바탕화면에 새 폴더 생성**
  - 폴더 이름을 `01_HTML`로 지정합니다.

- **VSCode에서 폴더 열기**
  - VSCode 상단 메뉴에서 **File** $\rightarrow$ **Open Folder**를 선택합니다.
  - 생성한 `01_HTML` 폴더를 선택하고 열어줍니다.

- **보안 안내 창 확인**
  - "Do you trust the authors of the files in this folder?"라는 안내 창이 뜨면 "Yes, I trust the authors"를 클릭합니다.

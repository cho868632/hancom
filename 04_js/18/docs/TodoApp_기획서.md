# 바닐라 JS Todo APP 기획서

## 1. 개요

- **목적**: 프레임워크 없이 바닐라 HTML/CSS/JS만으로 CRUD와 캘린더 기반 일정 관리를 지원하는 투두 리스트 앱을 구현한다.
- **범위**: 할 일 추가/수정/삭제/완료 처리 같은 핵심 CRUD 동작에 더해, 날짜별로 할 일을 등록하고 확인할 수 있는 달력 뷰를 함께 제공한다.
- **배포**: Netlify에 정적 사이트로 배포한다. 별도 서버(JSON Server 등)를 두면 배포 환경에서 데이터가 유지되지 않으므로, 데이터 저장은 `localStorage`를 사용한다.
- ~~**확장 여지**: 여유가 되면 캘린더 기능 추가를 고려한다.~~

## 2. 레이아웃 구성

화면 전체를 사이드바 + 메인 영역(달력 + 투두리스트)으로 나눠서 사용한다.

### 사이드바

- 다크 모드 / 라이트 모드를 전환하는 버튼
- 전체 / 해야 할 일 / 완료 개수를 보여주는 통계 카드
- 진행 중 / 완료 상태별로 필터링할 수 있는 네비게이션
- 완료된 항목을 한 번에 삭제하는 버튼

### 메인 영역

- 상단에 할 일을 입력하는 인풋창과 추가 버튼
- 본문은 `main-body` 그리드로 달력 섹션과 할 일 목록 섹션을 좌우 분할
  - **달력 섹션**: 이전/다음 달 이동 버튼, "오늘" 버튼, 요일 헤더(일 = 빨강, 토 = 파랑), 월 날짜 그리드
    - 날짜를 클릭하면 해당 날짜가 `selectedDate`로 선택되어 새로 추가하는 할 일에 그 날짜가 부여된다
    - 할 일이 등록된 날짜에는 점(dot) 인디케이터가 표시된다
  - **할 일 목록 섹션**: 항목별 할 일 텍스트 + 등록 날짜 표시, 빈 상태 안내 텍스트("표시할 할 일이 없어요"), 좌측 하단에 진행 중 개수 표시
- 반응형: 1200px 이하에서는 달력/목록 비율 축소(5:5), 900px 이하에서는 세로로 스택, 780px 이하에서는 사이드바가 상단 가로 바로 전환된다

## 3. 데이터 구조

투두 아이템(`todo`) 하나는 다음 필드를 가진다.

| 필드      | 타입    | 설명                                                                              |
| --------- | ------- | --------------------------------------------------------------------------------- |
| `id`      | number  | 고유값. `Date.now()`로 생성                                                       |
| `text`    | string  | 할 일 내용                                                                        |
| `checked` | boolean | 완료 여부. 기본값 `false`                                                         |
| `date`    | string  | 할 일이 속한 날짜 (`YYYY-MM-DD`). 추가 시점의 `selectedDate`(기본값: 오늘)로 지정 |

전체 목록은 배열 형태로 `localStorage`에 저장하고, 페이지 로드 시 불러온다.

## 4. 핵심 기능

- **추가**: 인풋에 텍스트 입력 후 제출하면 현재 선택된 날짜(`selectedDate`)를 가진 새 `todo`가 목록에 추가된다.
- **완료 토글**: 각 항목의 체크박스로 `checked` 상태를 전환한다.
- **삭제**: 항목 단위 삭제, 완료된 항목 일괄 삭제를 지원한다.
- **수정**: 할 일 텍스트를 인라인으로 수정한다.
- **필터링**: 전체 / 진행 중 / 완료 상태로 목록을 필터링한다.
- **빈 상태 처리**: 필터 결과가 없을 때 안내 텍스트를 노출한다.
- **테마 전환**: 다크/라이트 모드를 토글하고 상태를 유지한다.
- **달력**:
  - 월 단위로 이전/다음 달 이동, "오늘" 버튼으로 현재 월/날짜로 즉시 복귀
  - 날짜를 클릭해 `selectedDate`를 지정 (새 할 일의 기본 날짜로 사용됨)
  - 할 일이 있는 날짜에 점 인디케이터 표시

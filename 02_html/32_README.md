# 개발자도구 (F12) — 화면 뒤를 들여다보는 꿀팁

브라우저에 숨어 있는 개발자도구를 활용하면 화면 뒤의 HTML·CSS를 모두 확인하고, 값을 실시간으로 변경해 미리볼 수 있습니다. 별도의 설치 없이 크롬과 엣지에 기본 내장되어 있는 강력한 실전 도구입니다.

---

## ⌨️ 여는 법

- **단축키 이용:** `F12` 누르기 _(Mac은 `Cmd` + `Opt` + `I`)_
- **마우스 이용:** 화면 아무 곳이나 마우스 우클릭 → **[검사]** 선택

---

## 💡 실전 활용 꿀팁

### 1️⃣ 요소 콕 짚어내기

- **화살표(검사) 아이콘**을 클릭한 후, 화면의 특정 요소에 마우스를 올리면 해당 요소가 어떤 HTML 태그로 구성되어 있는지 즉시 찾아줍니다.

### 2️⃣ HTML 구조 파악 (Elements 탭)

- **Elements 탭**은 웹 페이지의 전체적인 HTML 구조를 보여줍니다. 코드 줄을 클릭하면 화면에서 그 코드가 담당하는 영역이 반짝이며 표시됩니다.

### 3️⃣ 디자인 실시간 미리보기

- 텍스트나 색상 값을 **더블클릭**하여 원하는 값으로 수정해 보세요. 화면에 즉시 반영되므로 디자인을 자유롭게 실험해볼 수 있습니다.

### 4️⃣ 에러 잡기 (Console 탭)

- **Console 탭**에 표시되는 **빨간 글씨**는 웹 페이지의 에러 메시지입니다. 개발 중 어디서 오류가 발생했는지 결정적인 힌트를 제공합니다.

> ⚠️ **주의사항**
> 개발자도구에서 변경한 모든 값은 현재 브라우저 화면에만 **임시로 반영**됩니다. 실제 소스 파일은 변경되지 않으며, **새로고침하면 원래대로 원상복구**되므로 두려워하지 말고 마음껏 실험하셔도 안전합니다.

---

## 🚀 가장 빠른 웹 공부법

> **"남이 잘 만든 사이트에서 F12를 열어 구조를 구경하는 것"**
> 트렌디하고 완성도 높은 사이트들이 코드를 어떻게 배치했는지 눈으로 확인하는 것이 실력을 키우는 가장 빠른 지름길입니다.
